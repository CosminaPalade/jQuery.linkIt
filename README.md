LinkIt jQuery Plugin
================================
**LinkIt** is a simple jQuery plugin to attach links to elements.

Version
---------------------------------
0.1.0

Usage
---------------------------------
```js
url("data/test.html");

=> "data/test.html?10538358428943"


url("data/test.php?foo=bar");

=> "data/test.php?foo=bar&10538358345554"
```
